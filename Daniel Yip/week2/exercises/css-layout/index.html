static (default):
the position is whatever the browser calculates fot this element in the page flow; you cant change it with top/left CSS offsets
relative:
Offsets do work, and are calculated relative to the default (static) position in the browser's page flow.
(parent of absolute)
absolute:
Offsets work, and are calculated relative to the nearest containing elemnt (ancestor) which has a specific non-static position value... otherwise relative to the body tag, i.e. whole document.
fixed:
Offsets are calculated realtve to the entire browser window (viewport) - i.e. the element does not move when you scroll te document
