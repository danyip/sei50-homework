static (default):
The position is whatever the browser calculates for this element in the page flow; you can't change it with top/left CSS offsets.
relative:
Offsets do work, and are calculated relative to the default (static) position in the browser's page flow.
absolute:
Offsets do work, and are calculates relative to the nearest containing element (ancestor) which has a specific non-static position value. Otherwise, this becomes relative to the "body" tag, i.e whole document.

Offsets are calculated relative to the entire browser window (viewport) - i.e. the element does NOT move when you scroll the document.
