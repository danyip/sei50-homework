absolute :
Offsets work, and are calculated relative to the nearest containing element (ancestor) which has a specific non-static position value...otherwise relative to the "body" tag, i.e. whole document.
fixed:
Offsets are calculated ralative to the entire browser window (viewpoint) - i.e. the element does NOT move when you scroll the document.
