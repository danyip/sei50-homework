CSS layout

Let's do this

static (default):
the position is whatever the browser calculates for this element in the page flow; you can't change it with top/left css offsets.
position relative:
offsets do work, and are calculated relative to the default (static) position in browser's page flow
(parent of absolute)
position absolute:
offsets work, and are calculated relative to the nearest containing element(ancestor) which has specific non-static position value... otherwise relative to the "body" tag, i.e. whole document.
fixed:
offsets are calculated relative to the entire browser window (viewport) - i.e. the element does NOT move when you scroll the document
